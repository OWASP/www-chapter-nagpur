## Resources

A curated collection of free security resources for the Nagpur OWASP community.

---

### OWASP Core Projects

| Resource | Description |
|---|---|
| [OWASP Top 10](https://owasp.org/www-project-top-ten/) | Standard awareness document for web application security risks |
| [OWASP API Security Top 10](https://owasp.org/www-project-api-security/) | Top 10 API security risks every developer should know |
| [OWASP Mobile Top 10](https://owasp.org/www-project-mobile-top-10/) | Top security risks in mobile applications |
| [OWASP Testing Guide](https://owasp.org/www-project-web-security-testing-guide/) | Comprehensive web application penetration testing guide |
| [OWASP ASVS](https://owasp.org/www-project-application-security-verification-standard/) | Application Security Verification Standard |
| [OWASP Cheat Sheet Series](https://cheatsheetseries.owasp.org/) | Quick reference guides for securing applications |
| [OWASP ZAP](https://www.zaproxy.org/) | Free, open-source web application security scanner |
| [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) | Intentionally insecure app for security training and CTFs |
